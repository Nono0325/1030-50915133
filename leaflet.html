Leaflet 地圖展示

Leaflet 官網

展示 Leaflet 用於創建互動地圖。

範例程式碼


    // 創建地圖並設置視圖
    const map = L.map('map').setView([25.033, 121.5654], 10);

    // 添加地圖圖磚
    L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
        maxZoom: 30
    }).addTo(map);

    // 在地圖上添加一個標記
    const marker = L.marker([25.033, 121.5654]).addTo(map);
    marker.bindPopup('這是台北!').openPopup(); // 添加彈出框
