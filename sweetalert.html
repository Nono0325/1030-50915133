SweetAlert2 彈窗展示

SweetAlert2 官網

使用 SweetAlert2 彈窗展示美觀的提示框。

範例程式碼


            <script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>
            <button onclick="showAlert()">展示基本彈窗</button>
            <script>
                function showAlert() {
                    Swal.fire('Hello World!', '這是一個 SweetAlert2 彈窗', 'success');
                }
            </script>
